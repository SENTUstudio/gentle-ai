## When to Use

- When source data is stored in S3 buckets (raw data landing zone)
- When incremental loading is needed (watermark-based)
- When source files are CSV, JSON, or other formats in S3
- When Glue Catalog table doesn't exist yet (landing to refined zone)
- During `sdd-design` phase for S3-based ETL approach

## Critical Patterns

### Incremental Watermark Pattern

The watermark pattern prevents re-processing already-loaded data:

```
1. get_last_load_timestamp(database, table) → last_load_time
2. get_s3_whitelist(bucket, prefix, pattern, last_load_time) → list of new files
3. Read only new files (incremental)
4. Write to partitioned output
5. Update last_load_timestamp
```

### S3 Output Partitioning
Always partition output by `year/month/day` for query efficiency:

```python
df_final.write.mode("append").partitionBy("year", "month", "day").parquet(S3_OUTPUT_PATH)
```

### Glue Catalog Sync
After writing, repair the Athena/Glue table to pick up new partitions:

```python
import awswrangler as wr
wr.athena.repair_table(table=GLUE_TABLE, database=GLUE_DATABASE)
```

### Schema Extraction Phase
When extracting schema (Phase 1 of three-phase workflow):
- **Comment out** write sections (S3 write + catalog sync)
- **Add** `df_final.printSchema()` or inspect `df.dtypes`
- Purpose: Document column names and types before creating table

## Code Examples

### Glue Job: ETL from S3

```python
# ============================================================================
# Glue Job: ETL_<table>_carga
# source: S3
# author: generated by gentle-ai
# ============================================================================

import sys
import os
import logging
from datetime import datetime, date
from pyspark.context import SparkContext
from awsglue.context import GlueContext
from awsglue.dynamicframe import DynamicFrame
from awsglue.job import Job
from awsglue.utils import getResolvedOptions
from pyspark.sql.functions import lit
import fnmatch
import awswrangler as wr

# Configuration
BUCKET_RAW = os.getenv("BUCKET_RAW", "toyota-chile-raw-data")
BUCKET = os.getenv("BUCKET", "toyota-chile-refined")
S3_FILE_PATTERN = "Output/*.csv"
S3_OUTPUT_PATH = f"s3://{BUCKET}/<table_name>/"

# Glue Catalog configuration
GLUE_DATABASE = os.getenv("GLUE_DATABASE", "toyota_chile")
# Encoding — should come from study-file output; defaults to UTF-8
ENCODING = os.getenv("ENCODING", "UTF-8")

def clean_column_name(name: str) -> str:
    """Normalize column names: lowercase, underscores."""
    import re
    name = re.sub(r'[^a-zA-Z0-9]', '_', name)
    return name.lower()

def get_last_load_timestamp(database: str, table: str) -> str:
    """Get last load timestamp from Glue Catalog or return epoch."""
    try:
        result = wr.athena.read_sql_query(
            sql=f"SELECT MAX(last_load_timestamp) as ts FROM {database}.{table}_metadata",
            database=database
        )
        return result['ts'].iloc[0] if len(result) > 0 else "1970-01-01"
    except Exception:
        logging.warning("Failed to get last load timestamp, using epoch")
        return "1970-01-01"

def get_s3_whitelist(bucket: str, prefix: str, pattern: str, last_load: str) -> list:
    """Return S3 keys matching pattern modified after last_load timestamp."""
    import boto3
    s3 = boto3.client('s3')
    files = []
    paginator = s3.get_paginator('list_objects_v2')
    for page in paginator.paginate(Bucket=bucket, Prefix=prefix):
        for obj in page.get('Contents', []):
            if obj['LastModified'].isoformat() > last_load:
                if fnmatch.fnmatch(obj['Key'], f"{prefix}{pattern}"):
                    files.append(f"s3://{bucket}/{obj['Key']}")
    return files

# ============================================================================
# INIT
# ============================================================================
args = getResolvedOptions(sys.argv, ["JOB_NAME", "BUCKET_RAW", "BUCKET_STG"])
sc = SparkContext.getOrCreate()
glueContext = GlueContext(sc)
spark = glueContext.spark_session
job = Job(glueContext)
job.init(args["JOB_NAME"], args)

# ============================================================================
# READ (with watermark for incremental)
# ============================================================================
last_load = get_last_load_timestamp(GLUE_DATABASE, GLUE_TABLE)
s3_files = get_s3_whitelist(BUCKET_RAW, "Output/", S3_FILE_PATTERN, last_load)

if not s3_files:
    print("No new files to process")
    job.commit()
    sys.exit(0)

# Read with detected parameters from study-file
df = spark.read.csv(
    s3_files,
    header=True,
    sep=";",  # Toyota Chile uses semicolon
    encoding=ENCODING,  # From study-file output
    inferSchema=False
)

# ============================================================================
# TRANSFORM
# ============================================================================
# Clean column names
for col in df.columns:
    df = df.withColumnRenamed(col, clean_column_name(col))

# Add processing metadata
df = df.withColumn("fecha_proceso", lit(datetime.now().strftime("%Y-%m-%d")))
df = df.withColumn("year", lit(date.today().strftime("%Y")))
df = df.withColumn("month", lit(date.today().strftime("%m")))
df = df.withColumn("day", lit(date.today().strftime("%d")))

# Apply transformation logic (from sql-from-logic output)
# df_final = df.transform(transformation_function)
df_final = df  # Placeholder

# ============================================================================
# WRITE (PARTITIONED) — Commented for schema extraction phase
# ============================================================================
# df_final.write.mode("append").partitionBy("year", "month", "day").parquet(S3_OUTPUT_PATH)
# wr.athena.repair_table(table=GLUE_TABLE, database=GLUE_DATABASE)

# ============================================================================
# SCHEMA EXTRACTION (uncomment for Phase 1)
# ============================================================================
df_final.printSchema()
print("Schema extraction complete")

# ============================================================================
# COMMIT — Always call job.commit() before exiting
# ============================================================================
job.commit()
```

```bash

    --template-file template.yaml \

aws cloudformation deploy \
    --template-file packaged.yaml \
    --stack-name ETLTableName \
    --parameter-overrides Environment=dev
```
